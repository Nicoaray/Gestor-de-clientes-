# Gestor-de-clientes-
Proyecto de API REST para la gestión de consultas médicas. Permite registrar y autenticar usuarios, administrar consultas mediante operaciones CRUD y proteger los endpoints con JWT. Implementa arquitectura en capas (Controller, Service, Repository), uso de DTOs, validaciones y persistencia en base de datos relacional.

Sistema de Gestión de Consultas Médicas (API REST)

Una API que permita:

Registrar usuarios (médicos / pacientes)/
Crear consultas médicas/
Listar consultas/
Modificar consultas/
Eliminar consultas/
Proteger endpoints con autenticación JWT

¿Qué problema resuelve?
Permite administrar turnos y consultas médicas de manera digital, separando:

Gestión de usuarios/
Gestión de consultas/
Seguridad/
Persistencia en base de datos
